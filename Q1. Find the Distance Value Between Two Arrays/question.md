## Issue Description
Given two arrays, count elements in the first array whose distance from every element in the second array is greater than a given value.  
Encourages thinking beyond brute-force comparison.

## Acceptance Criteria
- Correct count  
- Efficient handling of large inputs  
- Proper use of sorting or binary search
