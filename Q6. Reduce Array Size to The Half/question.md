## Issue Description
Given an integer array, remove all occurrences of selected integers such that at least half of the array elements are removed.
The goal is to minimize the number of selected integers.
This problem focuses on frequency counting and greedy selection.

## Acceptance Criteria
- Correct frequency calculation  
- Greedy removal strategy  
- Minimum set size returned
