## Issue Description
Find the length of the shortest non-empty subarray whose sum is at least K.
This problem requires prefix sums and a monotonic deque for optimization.

## Acceptance Criteria
- Prefix sum usage  
- Monotonic deque logic  
- O(n) time complexity
