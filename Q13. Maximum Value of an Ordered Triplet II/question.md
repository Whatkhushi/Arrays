## Issue Description
Find the maximum value of (nums[i] - nums[j]) * nums[k] such that i < j < k.
The solution requires careful traversal and tracking values.

## Acceptance Criteria
- Correct ordering constraint  
- Efficient traversal  
- Returns 0 if no valid triplet
